Auditor de força de senhas desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades
- Análise de força de senha
- Verificação de critérios de segurança
- Gerador de senha segura
- Interface responsiva

## Tecnologias
- HTML5
- CSS3
- JavaScript

## Objetivo
Demonstrar fundamentos de segurança, validação e experiência do usuário para portfólio em cibersegurança.
